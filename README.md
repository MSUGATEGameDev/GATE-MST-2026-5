# Portal Jumper
**By:** Wyatt Buehner

### Instructor Thoughts
This was such a unique idea. It was really cool that you built a new mechanic in the game and were able to do it all by only taking advantage of the pre-existing tools. Platforming, maze navigation, and puzzle-solving between parallel universes was really fun. Very reminiscent of great puzzle games like Portal.

### Tweaks we made post-class
Outside of the stuff we fixed for everyone, we only had one thing we had to tweak, and that’s that the button that leads to the main maze was easy to accidentally press when the door wasn’t active, so we moved it to the same layer as the door. 

### Moving Forward
You could simplify your jump mechanic by adding a few lines of code to the jump function in the player controller to activate the show/hide actions instead of placing invisible buttons around the map everywhere you might jump. Other than that, it will just be cool to see what additional puzzles you come up with to make use of the game’s mechanics.
